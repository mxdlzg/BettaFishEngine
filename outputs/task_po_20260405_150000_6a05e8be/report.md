基于当前技术限制与数据现状，提出以下具体建议：

### 7.1 技术修复（优先级：最高）

2.  **依赖安装**：在 `InsightEngine` 环境中安装缺失的 `torch` 模块（错误日志：`No module named 'torch'`）。
3.  **功能验证**：修复后需进行回归测试，确保 Query、Media、Insight 三大引擎能正常输出最近 7 天的数据。

2.  **实时接入**：建立备用数据接口，确保在核心引擎故障时，至少能获取基础新闻标题与来源数量。
